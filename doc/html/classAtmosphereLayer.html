void AtmosphereLayer::InitScreen (  ) 

Initialization for the Atmosphere Layer.

Set up parameters for generating a simulated atmospheric phase screen. Assumes Kolmogorov spectrum of atmospheric disturbances [c.f. Tatarski 1961,1971], s.t. the phase structure function is given by D(r) = < [ phi( r' ) - phi( r' + r ) ]**2 > = 6.88 * ( r / r0 )**5/3 where r0 is the Fried parameter.
