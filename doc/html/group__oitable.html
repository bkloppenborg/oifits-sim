Detailed Description

This module is derived from the "OIFITS example software in C". It provides the same Application Programming Interface (API) as the example software, with the addition of a set of free_oi_*() functions.

A higher-level API, containing functions to read and write an entire OIFITS file, is provided by the oifile module.
